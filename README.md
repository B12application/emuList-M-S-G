# B12 - Kişisel Medya ve Planlayıcı

`B12`, film, dizi, oyun ve kitap listelerini tek yerde tutmak için geliştirilmiş bir kişisel takip uygulamasıdır.  
Aynı zamanda geliştirici çalışma düzeni için takvim/planlayıcı bölümü de içerir.

## Neler Var?

- Film, dizi, oyun ve kitap ekleme
- İzledim/okudum/oynadım takibi
- Favori ve öneri sistemi
- Takvim/planlayıcı ile günlük düzen
- Profil, ayarlar ve bildirim alanları

## Teknolojiler

- React
- TypeScript
- Vite
- Tailwind CSS
- Firebase

## Kurulum

1. Projeyi indir:

```bash
git clone <repo-url>
cd emuList-M-S-G
```

2. Paketleri kur:

```bash
npm install
```

3. Geliştirme sunucusunu başlat:

```bash
npm run dev
```

4. Tarayıcıdan aç:

```text
http://localhost:5173
```

## Build Alma

```bash
npm run build
```

Build çıktısı `dist` klasörüne alınır.

## Notlar

- Uygulama giriş/profil akışında Firebase servislerini kullanır.
- Ortam değişkenleri için projedeki yapılandırma dosyalarını kontrol edin.
